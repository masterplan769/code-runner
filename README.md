# Code Runner

**Offline browser-based code runner** for JavaScript, Python, and Lua.  
Run code instantly, organize scripts in tabs, and enjoy a clean, modern interface with dark/light themes and drag & drop file support. Perfect for learning, testing snippets, or experimenting with multiple languages directly in your browser.
